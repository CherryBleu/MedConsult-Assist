# 全栈契约矩阵

> 用于机器检查 `docs/接口文档.md`、前端 API 调用、后端 Controller、数据表与测试文件是否同向。运行：`python scripts/review/verify_contract_matrix.py`。

## 状态说明

| 状态 | 含义 |
| --- | --- |
| `aligned` | docs、前端、后端、数据表和测试文件均有命中证据 |
| `aligned-service-test` | 端点契约命中，测试覆盖在 service/flow 层而非 Controller 专测 |
| `docs-gap` | 代码或前端已存在，但 docs 仍需继续补完整语义 |
| `test-gap` | docs/前后端契约命中，但缺少针对该端点的自动化测试 |
| `auth-gap` | 端点存在，但权限收口仍需单独治理 |
| `internal-docs-gap` | 内部接口已落地，需补内部契约文档或身份策略说明 |
